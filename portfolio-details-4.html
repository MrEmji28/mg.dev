Know Your Neighborhood

  • Category: Web Application
  • Project date: June, 2023
  • Project Type : Individual

I developed this application using React.js, a popular JavaScript library for building user interfaces, alongside a REST API for efficient data communication between the front-end and back-end components. To enhance user experience and security, I leveraged both the Meta API and Google Gmail API for streamlined login processes. This not only offers users the convenience of logging in with their existing accounts but also ensures robust authentication protocols.Moreover, integrating the Google Maps API allowed for dynamic mapping functionalities within the application, enabling users to explore and visualize the geographical layout of the Know Your Neighborhood area with ease. This feature enhances the overall usability and interactivity of the platform, providing valuable insights into the neighborhood's geographical context.
