AAA Hosting Services Web Application

  • Category: Web Application
  • Project type: Individual
  • Project date: August 2023

The AAA Hosting Services is a web application that I developed using Liferay to provide hosting services online. This web application enables users to register and order hosting services. Additionally, it allows the admin to view, approve, delete, or update user orders and memberships.

I developed this web application's user interface using Liferay Developer Studio. For the registration and customer portal, I utilized Spring Boot to create its backend functionality. This combination of technologies allowed for the seamless integration of frontend and backend components, ensuring smooth user experiences and efficient management of customer data. Additionally, leveraging Spring Boot provided scalability and flexibility for future enhancements and feature additions.
